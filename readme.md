# User Search Card Web App

A simple web application built using HTML, CSS, and JavaScript that allows users to search and filter user cards by name in real time.

## Features

- Real-time user search
- Responsive card design
- Fast and lightweight
- Dynamic filtering using JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript
